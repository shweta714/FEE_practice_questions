This page demonstrates how to add a favicon to an HTML document.

About Favicons

A favicon is a small icon that appears in:

  • Browser tabs
  • Bookmarks
  • Browser history
  • Desktop shortcuts

How to Add a Favicon

Add the following tag in the <head> section of your HTML:

<link rel="icon" type="image/x-icon" href="favicon.ico">
